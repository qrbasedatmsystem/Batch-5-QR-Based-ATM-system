Steps to execute the project :


1)	Initialize the database in Mobile Application.
2)	If the user is new then they should register with Name, email id and Phone number. Or if they are already a registered user then they can login directly with email id and password.
3)	After successful login the user should enter the amount to be withdrawn and press the generate QR code button.
4)	QR code will be generated.
5)	The generated QR code should be scanned by the camera in ATM then it will be decoded.
6)	Then scan the fingerprint should be scanned. If the fingerprint is not registered before or if it is not scanned properly then the user should rescan the fingerprint.
